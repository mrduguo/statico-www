CSS

CSS Minification

CSS will be minified with clean-css. You may pass additional minification options by cleancssOptions.

CSS Aggregation

CSS files will be aggregated automatically if there is no other html element between the CSS link tag. CSS @import rule will treated as aggregation to inline to the parent CSS file. Any sio annotation will start a new aggregation. {% highlight html %} {% endhighlight %}

Less CSS

Less will handled transparently by lessc. You may pass additional Less CSS options by lessOptions. To support Less CSS development locally, you may include the less lib and supported config with sio-local-development-only annotation, those file will be ignored during optimization. e.g. {% highlight html %} {% endhighlight %}
