HTML

File with extension of html, htm, inc, php, asp, aspx or jsp will be treated as HTML document. HTML document will be minified and scanned for CSS and JavaScript references for further optimization.

annotation: sio-use-cdn

Serve the referenced resources from CDN. e.g. {% highlight html %} {% endhighlight %}

annotation: sio-local-development-only

Ignore the file during optimization. e.g. {% highlight html %} {% endhighlight %}

annotation: sio-ref-id

Expose the optimized resource reference for further use to store in refs file.
