JavaScript Minification

JavaScript will be minified with uglifyjs. You may pass additional minification options by uglifyjsOptions.

JavaScript Aggregation

JavaScript files will be aggregated automatically if there is no other html element between the script tag. A sio-ref-id annotation will start a new aggregation. {% highlight html %} {% endhighlight %}

annotation: sio-js-source-map
